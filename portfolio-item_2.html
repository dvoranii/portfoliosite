OpenWeather API

Weather around the world

This application allows users to enter a city name and display the current weather at that location. This is done using the OpenWeatherMap API.

OpenWeatherMap requires a pro membership for SSL support, and therefore cannot be interfaced with live on GitHub pages without cost. To view the app for free, you may clone the GitHub repository, or check out the video demonstration below:
